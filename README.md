# Aadhaar Lifecycle Intelligence (ALI)

This repository contains the reproducible analysis submitted for the **UIDAI Data Hackathon 2026**.

## Overview

Aadhaar Lifecycle Intelligence (ALI) integrates Aadhaar enrolment, demographic update, and biometric update datasets to derive lifecycle-based indicators and machine learning–driven risk insights for governance and administrative planning.

## Contents

- **analysis_uidai_final.ipynb** – End-to-end reproducible analysis notebook  
- **Charts** – Visualisations including EGI, USI, BTP, Risk Quadrant, ML Risk Clusters, and Feature Importance  
- **Insights** – Lifecycle indicators and unsupervised machine learning–based risk segmentation  

## How to Run

1. Download the Aadhaar datasets from the official UIDAI Data Hackathon portal.  
2. Place the datasets in the appropriate `data/` folder structure.  
3. Open `analysis_uidai_final.ipynb` in Jupyter.  
4. Restart & Run All to reproduce all results.

## Usage

- Clone the repository  
- Download the Aadhaar datasets (from the official hackathon portal)  
- Place them under the `data/` directory  
- Run the notebook end-to-end  

## Note

Datasets are not included in this repository due to size constraints and must be downloaded from the official UIDAI Data Hackathon portal:
